Espejo de Árbol Binario

Visualización del proceso de inversión de un árbol binario usando recursión

Definición del Problema

El algoritmo de espejo invierte un árbol binario, intercambiando el hijo izquierdo por el derecho en cada nodo recursivamente.

Algoritmo

def mirror_tree(root):
    if root is None:
        return None
    root.left, root.right = root.right, root.left
    mirror_tree(root.left)
    mirror_tree(root.right)
    return root

La función utiliza recursión para recorrer el árbol e intercambiar los hijos izquierdo y derecho de cada nodo.

Casos de Prueba

  • Árbol completo: se invierten los subárboles izquierdo y derecho.
  • Árbol vacío: permanece vacío tras aplicar el espejo.
  • Árbol con un solo nodo: no cambia tras aplicar el espejo.
