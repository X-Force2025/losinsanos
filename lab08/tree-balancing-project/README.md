# Tree Balancing Project

This project implements a function that balances a Binary Search Tree (BST) without modifying the inorder traversal.

## Files
- `balance_bst.py`: Contains the core implementation for BST and the balance function.
- `test_balance_bst.py`: Contains test cases to validate the implementation.

## How to Run
Run `test_balance_bst.py` to execute the test cases:

```bash
python test_balance_bst.py
```

Enjoy!
